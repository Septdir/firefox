## Installation
1. Download the latest version
2. Unpack the archive
3. Move the contents of the chrome folder to `%appdata%\Mozilla\Firefox\Profiles\XXXXXXXX\chrome` where `XXXXXXXX` is profile name
4. Restart the browser
